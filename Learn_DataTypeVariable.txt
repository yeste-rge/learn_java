资料形态与变数

#注解
写给人看的文字, 执行时会忽略

exm.
→单行注解范例:
//注解内容

→多行注解范例:
/*
    第一行注解
    第二行注解
*/

##资料形态 Data Types
Java 程式中可以操作的各种资料
exm.
→整数: byte,short,int,long
→浮点数: float,double
→布林值： boolean
→字元: char
→字串: String

#整数
代表各种范围的整数
exm.
→byte: -128~127
→short: -32768~32767
→int: -2147483648~2147483647
→long:
-9223372036854775808~
9223372036854775807

#浮点数
代表各种精准度的小数
exm.
→float:精准到小数点下7位数
→double:精准到小数点下15位数
